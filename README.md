# 📘 C Programming Lab Assignments

A collection of C programs covering arrays, pointers, functions, and pattern logic.


---

## 🔍 Programs Overview

### Q1 — Grade Calculator (`q1_Grade.c`)
Accepts marks of **5 subjects**, calculates total and percentage, then displays the grade.

| Percentage | Grade |
|------------|-------|
| > 70       | A     |
| > 60       | B     |
| > 50       | C     |
| > 40       | D     |
| < 35       | E     |
| Otherwise  | Fail  |

**Concepts used:** `if-else`, arithmetic operators, `scanf`/`printf`

---

### Q2 — 2D Array Average (`q2_Average.c`)
Takes **9 integers** as input into a **3×3 matrix**, then calculates and prints the average using a separate function.

**Concepts used:** 2D arrays, functions, nested loops, `float` division

---

### Q3 — Odd Elements (`Q3_odd_elment.c`)
Accepts an array of `n` integers and prints all **odd elements** using a user-defined function.

**Concepts used:** 1D arrays, functions, modulus operator, VLA (Variable Length Array)

---

### Q4 — Even Elements Using Pointer (`Q4_even.c`)
Accepts an array of `n` integers and prints all **even elements** using a **pointer**.

**Concepts used:** Pointers, pointer arithmetic, `*(ptr + i)` notation, VLA

---

### Q5 — Number Pattern (`Q5_pattern.c`)
Prints a right-aligned **staircase number pattern** starting from 10 down to 6.

**Output:**
```
                    10
                9   10
            8   9   10
        7   8   9   10
    6   7   8   9   10
```

**Concepts used:** Nested loops, space formatting, pattern logic
